# 🔥 SPARK Restaurant Website

موقع مطعم سبارك الكامل — بيتزا، كالزوني، كريب، رول، بطاطس، إضافات، مشروبات

## الملفات

| الملف | الوصف |
|-------|-------|
| `spark-restaurant.html` | الموقع الرئيسي الكامل |
| `spark-admin.html` | لوحة التحكم |

## كيفية الفتح محلياً

1. افتح VS Code
2. File → Open Folder → اختار مجلد `spark-website`
3. اضغط على `spark-restaurant.html` في الشريط الجانبي
4. اضغط كليك يمين → Open with Live Server (لو عندك extension)
5. أو اضغط على الملف مباشرة من File Explorer لفتحه في المتصفح

## المميزات

- 7 أقسام | 69 صنف كامل
- سلة تسوق + Checkout
- كود خصم: SPARK20 (خصم 20%)
- Dark/Light mode
- عربي / إنجليزي
- ريسبونسف على كل الشاشات
- لوحة تحكم كاملة (إضافة/تعديل/حذف منتجات)
